# Zahnarzt Adolf Roth

Jede sichtbare Seitensektion liegt einzeln unter `sections/`.

`src-layout.html` bestimmt Reihenfolge und globale Einbindung. `assets/css/styles.css` enthält globale Styles. `assets/js/main.js` enthält Interaktionen. `assets/js/tailwind-config.js` enthält Farben und Schriften.

Nach Änderungen:

1. `npm run build`
2. `npm run dev`
3. `http://localhost:4173` öffnen

`npm run dev` baut bei jedem Seitenaufruf automatisch neu.
